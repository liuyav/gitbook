1.5. docker命令

简单启动：docker run -p 80:80 -v $PWD/www:/user/share/nginx/html -d nginx 查看进程：docker ps 停止进程：docker stop pid 删除镜像：docker rm pid 拉取镜像：docker pull nginx 查看镜像：docker image nginx 制作镜像：docker build -t myimage . 启动定制镜像：docker run -p 80:80 -d myimage
