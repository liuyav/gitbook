osi参考模型

  2. 数据链路层
  3. 网络层
  4. 传输层
  5. 会话层
  6. 表示层

tcp/ip模式

  1. 以太网、无线（1、2）
  2. 网络层（3）
  3. 传输层（4）
  4. 应用层（5、6、7）

通常是在传输层与应用层进行编程

HTTP 状态码

  1. 1xx：已经接收，继续处理
  2. 2xx：成功
  3. 3xx：重定向
  4. 4xx：客户端错误
  5. 5xx：服务器错误

了解常用的状态码 200、400、401、403、404、500、503

跨域

  1. 简单请求：get、post跨域（后端加入可允许访问源）
  2. 预检请求：put、delete等都有options类型请求，设置header
  3. nginx 反向代理（含义）
